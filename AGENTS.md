# AGENTS.md

## 코드 컨벤션

- `*.kt` 파일이 수정되면 ktlint 포메팅을 맞춥니다. `./gradlew ktlintFormat`

### 코드 브레이싱

`if`, `for`, `while` 등 다음 코드가 한 줄 일 때에도 브레이서를 무조건 표기합니다.

- 잘못된 코드
    ```kotlin
    if (x == 0) return
    ```
- 올바른 코드
    ```kotlin
    if (x == 0) {
        return
    }
    ```

## 백엔드

#### 코틀린

- 하나의 `kt` 파일에는 하나의 클래스만 작성합니다.

## 테스트

- 개발자가 직접 테스트 합니다.
- 자동으로 테스트하거나 테스트 명령을 수행하지 않습니다.

## 기타

- 대화는 항상 한글로 합니다.
- 현재 작업 디렉토리에서 일어나는 커맨드 명령어는 물어보지 않고 바로 수행합니다.
- 현재 작업 디렉토리가 아닌 다른 곳에 파일이 생성/수정/삭제 되는 경우는 나에게 물어봅니다.
- 파일을 읽는 명령어는 물어보지 않고 바로 수행합니다.
